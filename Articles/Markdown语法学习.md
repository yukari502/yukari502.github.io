# Markdown 语法学习

这是一个 Markdown 语法学习文档

## 1. 多级标题

Markdown 支持六级标题，通过在行首添加不同数量的 `#` 符号来表示。

### 1.1 三级标题示例

```markdown
### 这是一个三级标题
```

#### 1.1.1 四级标题示例

```markdown
#### 这是一个四级标题
```

##### 1.1.1.1 五级标题示例

```markdown
##### 这是一个五级标题
```

###### 1.1.1.1.1 六级标题示例

```markdown
###### 这是一个六级标题
```

## 2. 文本样式

### 2.1 加粗

使用两个星号 `**` 或两个下划线 `__` 包裹文本可以使其加粗。

**示例：**

```markdown
**这是加粗的文本**
__这也是加粗的文本__
```

**效果：**
**这是加粗的文本**
**这也是加粗的文本**

### 2.2 斜体

使用一个星号 `*` 或一个下划线 `_` 包裹文本可以使其斜体。

**示例：**

```markdown
*这是斜体的文本*
_这也是斜体的文本_
```

**效果：**
*这是斜体的文本*
*这也是斜体的文本*

### 2.3 加粗斜体

同时使用三个星号 `***` 或三个下划线 `___` 包裹文本可以使其加粗并斜体。

**示例：**

```markdown
***这是加粗斜体的文本***
___这也是加粗斜体的文本___
```

**效果：**
***这是加粗斜体的文本***
***这也是加粗斜体的文本***

### 2.4 划掉 (删除线)

使用两个波浪线 `~~` 包裹文本可以使其显示删除线。

**示例：**

```markdown
~~这是被划掉的文本~~
```

**效果：**
~~这是被划掉的文本~~

## 3. 链接与图片

### 3.1 添加超链接

使用方括号 `[]` 包裹链接文本，后面紧跟圆括号 `()` 包裹链接地址。

**示例：**

```markdown
[访问 Codeium 官网](https://www.codeium.com)
```

**效果：**
[访问 Codeium 官网](https://www.codeium.com)

### 3.2 添加图片

图片语法与超链接类似，但在方括号前多一个感叹号 `!`。方括号内是图片的替代文本，圆括号内是图片地址（可以是本地路径或网络 URL）。

**示例：**

```markdown
![Markdown Logo](https://www.markdown.cn/images/logo.png)
```

**效果：**
![Markdown Logo](https://www.markdown.cn/images/logo.png)

## 4. 代码块

### 4.1 行内代码

使用反引号 `` ` `` 包裹文本可以表示行内代码。

**示例：**

```markdown
这是一个 `行内代码` 示例。
```

**效果：**
这是一个 `行内代码` 示例。

### 4.2 代码块 (Fenced Code Block)

使用三个反引号 ``` ` `` ` 包裹多行代码，并在第一个反引号后指定语言可以实现语法高亮。

**示例 (Python):**

````markdown
```python
def hello_world():
    print("Hello, Markdown!")

hello_world()
```
````

**效果：**

```python
def hello_world():
    print("Hello, Markdown!")

hello_world()
```

**示例 (JavaScript):**

````markdown
```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}
greet("Codeium");
```
````

**效果：**

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}
greet("Codeium");
```

## 5. 表格

使用 `|` 分隔列，使用 `-` 分隔表头和表体。冒号 `:` 可以控制对齐方式。

**示例：**

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 单元格1 | 单元格2  | 单元格3 |
| 单元格A | 单元格B  | 单元格C |
```

**效果：**

| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 单元格1 | 单元格2  | 单元格3 |
| 单元格A | 单元格B  | 单元格C |

## 6. 文本对齐 (使用 HTML)

Markdown 本身不直接支持文本对齐，但可以通过内嵌 HTML 标签来实现。

**示例：**

```markdown
<p align="left">这段文字将左对齐。</p>
<p align="center">这段文字将居中对齐。</p>
<p align="right">这段文字将右对齐。</p>
```

**效果：**
<p align="left">这段文字将左对齐。</p>
<p align="center">这段文字将居中对齐。</p>
<p align="right">这段文字将右对齐。</p>
