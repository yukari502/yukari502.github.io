# 群体与进化遗传分析

### 1. 章节核心内容归纳

本章主要探讨了基因在群体中的分布、传递规律以及影响基因频率变化的因素，核心围绕**Hardy-Weinberg定律**展开。

**1. 群体遗传学的基本概念**
*   **达尔文进化论**：基于变异、遗传和选择三个原则。
*   **孟德尔群体 (Mendelian population)**：指个体的集合，这些个体间通过有性生殖进行基因交流（随机交配），基因一代代传递。
*   **基因库 (Gene Pool)**：一个群体内所包含的基因总数。
*   **频率计算**：
    *   **基因型频率**：特定基因型个体数 / 总个体数。
    *   **等位基因频率**：某特定基因总数 / 等位基因总数。例如 $p = D + \frac{1}{2}H$。

**2. Hardy-Weinberg 定律 (遗传平衡定律)**
*   **内容**：在一个巨大的、随机交配的群体中，如果没有突变、选择、迁移和遗传漂变，基因频率和基因型频率将代代保持不变。
*   **平衡公式**：$(p+q)^2 = p^2 + 2pq + q^2 = 1$。
    *   $p^2$ (AA), $2pq$ (Aa), $q^2$ (aa)。
*   **特征**：
    *   无论起始频率如何，只要随机交配一代，即可达到平衡。
    *   杂合体频率 $H$ 在 $p=q=0.5$ 时达到最大值 0.5。
    *   $H^2 = 4DR$ (杂合体频率是两个纯合体频率乘积的平方根的两倍)。
*   **应用与扩展**：
    *   **MN血型**：通过基因型频率验证群体是否处于平衡状态（卡方检验）。
    *   **复等位基因 (ABO血型)**：$(p+q+r)^2=1$。
    *   **伴性遗传**：雌性($XX$)有三种基因型，符合 $p^2, 2pq, q^2$；雄性($XY$)是半合子，基因型频率直接等于基因频率 ($p, q$)。

**3. 影响遗传平衡的因素**
现实中不存在理想群体，以下因素会打破平衡：
*   **非随机交配 (Non-random mating)**：
    *   **近亲交配**：导致群体中**纯合体频率增加，杂合体频率降低**。
    *   **Wright定律**：引入近交系数 $F$，平衡时的频率变为 $D=p^2+Fpq$, $H=2pq(1-F)$, $R=q^2+Fpq$。
*   **自然选择 (Natural Selection)**：
    *   **适应值 (Fitness, W)**：个体生存和繁殖的能力。相对生殖率为1。
    *   **选择系数 (Selection Coefficient, s)**：$s = 1 - W$。表示被淘汰的程度。
    *   **工业黑化**：环境改变导致特定表型的适应值改变（如椒花蛾）。
*   **突变 (Mutation)**：提供进化的原材料，但单独作用时改变频率极慢。存在**突变-选择平衡**，如隐性致害基因频率 $q = \sqrt{u/s}$。
*   **迁移 (Migration/Gene Flow)**：个体迁入迁出导致基因交流。改变程度取决于迁移率 $m$ 和两群体频率差。
*   **遗传漂变 (Genetic Drift)**：
    *   小群体中由于抽样误差导致的基因频率随机波动。
    *   **奠基者效应 (Founder Effect)**：如Dunkers群体，少数个体建立新群体导致基因频率与原群体差异巨大。

**4. 自然群体中的遗传变异**
*   **多态性 (Polymorphism)**：群体中存在两个或多个高频率的等位基因。
*   **杂合度 (Heterozygosity)**：衡量变异程度的指标。
*   **变异层次**：
    *   表型变异（如蜗牛壳色）。
    *   染色体多态性（如倒位）。
    *   蛋白质/DNA多态性（如凝胶电泳检测同工酶）。

### 2. 重点名词解释

**群体遗传学 (Population Genetics, P):** 研究群体的遗传组成（基因频率和基因型频率）及其在时空上变化规律的遗传学分支学科。

**基因库 (Gene Pool, G):** 一个群体内所有个体所拥有的全部基因的总和。

**Hardy-Weinberg 定律 (Hardy-Weinberg Law, H):** 在一个巨大的、随机交配的群体中，若没有突变、选择、迁移和遗传漂变等因素的干扰，群体的基因频率和基因型频率将代代保持不变，处于遗传平衡状态。

**遗传漂变 (Genetic Drift, G):** 在小群体中，由于配子抽样的随机误差导致等位基因频率在世代间发生随机波动的现象。例如美国的Dunkers群体由于奠基者效应，其血型频率与德国原群体有显著差异。

**遗传多态性 (Genetic Polymorphism, G):** 在一个群体中，同一基因座上存在两个或两个以上频率较高的等位基因的现象；通常指频率最低的等位基因频率大于1%。

**适应值 (Fitness, W):** 又称适合度，指某一基因型的个体在特定环境中生存并传递基因给下一代的能力，通常用相对生殖率表示，最大值为1。

**选择系数 (Selection coefficient, S):** 表示某一基因型在选择作用下相对适应值降低的程度，即被淘汰的比例。公式为 $S = 1 - W$。

### 3. 重要实验/案例概述

**1. 工业黑化现象 (Industrial Melanism)**
*   **对象**：椒花蛾 (*Biston betularia*)。
*   **现象**：在工业革命前，浅色型蛾子占优势（地衣覆盖树干，浅色是保护色）。工业革命后，煤烟熏黑树干，黑色型蛾子频率急剧上升。
*   **机制**：**自然选择**。环境改变了不同基因型的**适应值 (W)**。在污染区，黑色型的 $W \approx 1$，浅色型易被鸟捕食，W较低；在非污染区则相反。这是自然选择导致基因频率定向改变的经典例证。

**2. Dunkers 种群研究 (奠基者效应)**
*   **背景**：18世纪从德国迁往美国的宗教隔离群体。
*   **发现**：Dunkers群体的血型频率（如A型血频率高）与德国原产地群体及美国周围人群均有显著差异。
*   **结论**：这是**遗传漂变**（具体为奠基者效应）的结果。由于起始迁移的个体数很少（50个家庭），其基因频率只是原群体的一个随机样本，导致了后代的频率偏移。

### 4. 章节作业 (Questions) 回答

根据文档末尾提供的章节作业 进行解答：

**题目背景：**
一个大的随机交配平衡群体中，隐性纯合体 (aa) 占 81% ($q^2 = 0.81$)。

**（1）若没有突变和选择，在下一代中显性纯合体，隐性纯合体以及杂合体的比例各为多少？**
*   **解析**：
    *   已知 $aa (q^2) = 0.81$，则 $q(a) = \sqrt{0.81} = 0.9$。
    *   则 $p(A) = 1 - 0.9 = 0.1$。
    *   根据H-W定律，若无干扰，频率代代不变。
*   **答案**：
    *   显性纯合体 (AA) = $p^2 = 0.1^2 = 0.01$ (1%)
    *   杂合体 (Aa) = $2pq = 2 \times 0.1 \times 0.9 = 0.18$ (18%)
    *   隐性纯合体 (aa) = $q^2 = 0.81$ (81%)

**（2）如果选择对隐性纯合体不利（S=0.5），则下一代a基因的频率是多少？**
*   **解析**：
    *   初始频率：$D(AA)=0.01, H(Aa)=0.18, R(aa)=0.81$。
    *   适应值：$W_{AA}=1, W_{Aa}=1, W_{aa}=1-S=0.5$。
    *   选择后各基因型比例贡献：
        *   $AA: 0.01 \times 1 = 0.01$
        *   $Aa: 0.18 \times 1 = 0.18$
        *   $aa: 0.81 \times 0.5 = 0.405$
    *   总和 (Mean Fitness, $\bar{W}$) = $0.01 + 0.18 + 0.405 = 0.595$。
    *   选择后 $aa$ 的相对频率 $R' = 0.405 / 0.595 \approx 0.6807$。
    *   选择后 $Aa$ 的相对频率 $H' = 0.18 / 0.595 \approx 0.3025$。
    *   下一代 $a$ 基因频率 $q_1 = R' + 1/2 H' = 0.6807 + 0.1512 = 0.8319$。
    *   (或者使用公式：$q_1 = \frac{q - sq^2}{1 - sq^2} = \frac{0.9 - 0.5 \times 0.81}{1 - 0.5 \times 0.81} = \frac{0.495}{0.595} \approx 0.83$)。
*   **答案**：约 0.83。

**（3）如果选择对隐性纯合体不利（S=1），需经过多少代的选择淘汰才能使a基因的频率减少为现在的1/10？**
*   **解析**：
    *   当 $S=1$ (完全淘汰隐性纯合体) 时，基因频率改变的公式为：$q_n = \frac{q_0}{1 + n q_0}$。
    *   已知 $q_0 = 0.9$。
    *   目标频率 $q_n = 0.9 \times \frac{1}{10} = 0.09$。
    *   代入公式：$0.09 = \frac{0.9}{1 + n(0.9)}$。
    *   $1 + 0.9n = \frac{0.9}{0.09} = 10$。
    *   $0.9n = 9 \implies n = 10$。
*   **答案**：需经过 10 代。

**（4）如果正向突变率为 $6 \times 10^{-7}$，回复突变率为 $6 \times 10^{-8}$，没有自然选择、迁移以及遗传漂变的影响，平衡时基因A和a的频率又是多少？**
*   **解析**：
    *   设正向突变率 ($A \to a$) 为 $u = 6 \times 10^{-7}$。
    *   设回复突变率 ($a \to A$) 为 $v = 6 \times 10^{-8}$。
    *   平衡时，$\Delta q = 0$，即 $up = vq$。
    *   平衡频率 $\hat{p} = \frac{v}{u+v}$，$\hat{q} = \frac{u}{u+v}$。
    *   计算：
        *   $u+v = 60 \times 10^{-8} + 6 \times 10^{-8} = 66 \times 10^{-8}$ (即 $6.6 \times 10^{-7}$)。
        *   $\hat{p}(A) = \frac{6 \times 10^{-8}}{66 \times 10^{-8}} = \frac{1}{11} \approx 0.09$。
        *   $\hat{q}(a) = \frac{60 \times 10^{-8}}{66 \times 10^{-8}} = \frac{10}{11} \approx 0.91$。
*   **答案**：平衡时基因 A 的频率约为 0.09，基因 a 的频率约为 0.91。
