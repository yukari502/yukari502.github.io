
# 植物生理学 水与植物细胞

本章的重点和难点在于细胞吸水的方式和机制，以及水势的概念。内容涵盖了水的生理生态功能、存在形式、理化性质、细胞水分关系及测定方法。

## 1. 水的生理生态功能与存在形式

### 1.1 水的生理生态功能

水在植物生命活动中具有多重重要意义：

* 原生质的主要组成部分（占 70% 至 90%）。
* 物质吸收和运输的良好溶剂。
* 许多代谢过程中的反应物质。
* 维持细胞膨压 (Maintain cell turgor)。
* 调节植物微气候，包括以水调温、以水调肥、以水调气、以水调湿。

### 1.2 水的存在形式

植物体内的水根据其流动性分为自由水和束缚水：

| 形式 | 解释说明 | 特点 | 来源 |
| :--- | :--- | :--- | :--- |
| **束缚水** (Bound water) | 被**原生质胶体** (protoplasm colloid) 吸附，不易流动。 | 不易流动和散失，含量稳定。维持原生质胶体的稳定性，参与植物对逆境的抵抗。 | |
| **自由水** (Free water) | 远离原生质胶体，可以自由流动。 | 含量变化大，与代谢强度相关。 | |

* **自由水/束缚水比值**：比值高意味着代谢强、抗性弱；比值低则代谢弱、抗性强。

## 2. 水的理化性质与移动机制

### 2.1 水的理化性质

水是**极性分子**，相邻水分子间通过静电吸引形成**氢键** (Hydrogen bond)。

| 性质 | 解释说明 | 来源 |
| :--- | :--- | :--- |
| **溶解特性** (Solubility characteristics) | 是生命活动中最重要的溶剂，也是已知溶剂中溶解范围最广的。 | |
| **比热** (Specific heat) | 水温不易改变，使整个生命活动在相对稳定的环境中进行。 | |
| **汽化热** (Heat of vaporization，蒸发潜热) | 在恒定温度下，分子从液相变为气相所需的热量。例如，25℃时，水的汽化潜热为 44 kJ/mol。 | |
| **内聚力** (Cohesion) | 水分子间的氢键使水分子相互吸引。 | |
| **黏着力** (Adhesion) | 水分子与固相（例如导管壁）之间的吸引力。 | |
| **抗拉强度** (Tensile strength) | 液体在断裂前能承受的最大拉力。内聚力赋予水较高的抗拉强度，小水柱可达 -30 MPa。 | |

**毛细现象 (Capillarity) 流程**：

1. **黏着力**使管壁周围的水上升，形成**凹月面** (concave lunar surface)。
2. **表面张力**倾向于减小表面积，形成向上的拉力 (up pull)。
3. **内聚力**将此拉力传递给水柱，导致水柱上升。
4. 水柱持续上升，直到向上的拉力等于水柱的重力为止。

### 2.2 水的移动方式

水的移动包括**扩散** (Diffusion)、**整体流动** (Bulk flow) 和**渗透** (Osmosis)。

* **渗透** (Osmosis)：水通过**半透膜** (semipermeable membrane) 从高 $\Psi_w$ 系统流向低 $\Psi_w$ 系统，由水势差驱动。

## 3. 水势及其组分

### 3.1 水势 (Water potential, $\Psi_w$)

**水势**是决定水流动方向的指标。

* **定义**：在相同的温度和压力下，水的化学势与纯水的化学势的差值除以水的偏摩尔体积。
* **规定**：纯水的水势最大，并设定为 **0**。
* **流动方向**：水从**高 $\Psi_w$** 流向**低 $\Psi_w$**。

### 3.2 水势的组分

植物细胞的总水势 ($\Psi_w$) 是由溶质势 ($\psi_s$)、压力势 ($\psi_p$)、基质势 ($\psi_m$) 和重力势 ($\psi_g$) 相加而成。

* **溶质势 (Solute potential，$\psi_s$)**：由于溶质颗粒存在导致的 $\Psi_w$ 下降值。可通过**范特霍夫方程** (Van’t Hoff Equation) 计算。溶质势是负值。
* **压力势 (Pressure potential，$\psi_p$)**：由于外部压力（如细胞壁）导致的 $\Psi_w$ 增加。
  * **正常细胞**：$\psi_p > 0$。
  * **质壁分离** (Plasmolysis) 时：$\psi_p = 0$。
  * **剧烈蒸腾**时：细胞壁产生巨大的张力或**负静水压** (negative hydrostatic pressure)，$\psi_p < 0$。
* **基质势 (Matric potential，$\psi_m$)**：由于胶体物质的**亲水性** (hydrophilicity) 和毛细管对水的束缚导致的 $\Psi_w$ 下降。
  * 在具液泡的细胞中，$\psi_m$ 影响极小，通常可忽略（约 -0.01 MPa）。
  * 在无液泡的**分生组织细胞**、**干种子**中，$\Psi_w$ 主要由 $\psi_m$ 决定。
* **重力势 (Gravitational potential，$\psi_g$)**：重力对 $\Psi_w$ 的影响，通常可忽略。

**特殊状态下的水势关系**：

1. **具液泡细胞**：$\Psi_w = \psi_s + \psi_p$。
2. **初生质壁分离细胞** (Initial plasmolysis cell)：$\Psi_w = \psi_s$ ($\psi_p = 0$)。
3. **水饱和细胞** (Water saturated cell)：$\Psi_w = 0$。

## 4. 细胞吸水机制与通路

### 4.1 细胞吸水形式

植物细胞吸收水分主要有三种形式：

1. **吸胀作用** (Imbibition) 引起的吸水（如风干种子）。
2. **渗透吸水** (Osmotic water absorption)（成熟细胞的主要吸水方式）。
3. **代谢吸水** (Metabolic water absorption)。

### 4.2 质壁分离 (Plasmolysis) 流程

质壁分离是指植物细胞的原生质体因液泡脱水而与细胞壁分离。

* **复原机制流程**：原生质体对蔗糖并非完全不透性。当发生质壁分离的细胞被置于蔗糖溶液中足够长时间后，蔗糖会**逐渐进入液泡**，导致细胞内的 $\Psi_w$ 和 $\psi_s$ 同时下降。水随后进入细胞，原生质体缓慢复原。

### 4.3 水孔蛋白 (Aquaporins, AQP) 机制

水进入细胞的途径之一是通过质膜上的**水孔蛋白**。

* **性质**：AQP 属于 **MIP** (Major Intrinsic Protein) **超家族**的通道蛋白。它们能**特异且高效**地允许 $\text{H}_2\text{O}$ 通过，同时转运其他小的中性分子 ($\text{CO}_2, \text{H}_2\text{O}_2$ 等)。
* **结构**：AQP 有六个跨膜结构域。孔隙由两个半孔隙 **NPA** ($\text{Asn-Pro-Ala}$) 组成，位于环 B 和 E 处。
* **转运机制流程**：
    1. $\text{H}_2\text{O}$ 以**单列**形式进入弯曲狭窄的通道。
    2. 通道的极性和耦合力有助于 $\text{H}_2\text{O}$ **旋转并穿过**。
    3. 通道内的**正电荷区域**会**阻止质子**的通过。
* **调控**：AQP 的**门控** (Gating) 活性和表达受复杂信号通路调控，包括**磷酸化**、pH、阳离子、活性氧 (ROS) 和植物激素等化学试剂。

## 5. 水势及其组分的测定方法

| 待测参数 | 测定方法 (Measurement Method) | 测定原理 (Principle) | 来源 |
| :--- | :--- | :--- | :--- |
| **水势 ($\Psi_w$)** | **干湿球温度计法** (Dry-wet bulb thermometer) | 基于溶液 $\Psi_w$ 降低会使水蒸气压降低（**溶液的依数性**）。 | |
| | **小液流法** (Small liquid flow method) | 当植物组织 $\Psi_w$ 与环境溶液 $\Psi_w$ 相等时，两者之间无水分迁移（渗透平衡）。 | |
| **压力势 ($\psi_p$)** | **压力室法** (Pressure bomb method) | 主要用于测定木质部 $\Psi_w$。 | |
| | **压力探针法** (Pressure probe method) | 通过压力使得细胞液刚好返回细胞来记录压力 ($\psi_p$)。 | |
| **溶质势 ($\psi_s$)** | **冰点下降法** (Freezing point falling method) | 基于溶质浓度增加，溶液冰点会降低（**溶液的依数性**）。 | |
